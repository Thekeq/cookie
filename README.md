# 🍪 Cookie Merge — Telegram GameFi Mini App

Merge-игра с кликером, уровнями, батл-пассом, рефералкой и покупками за Telegram Stars.

## Структура

```
bot/                  # aiogram: /start + оплата Stars + пуши
├── loader.py
├── notifier.py       # фоновые пуш-уведомления (энергия/ферма/стрик), макс 1 в ~20ч
└── handlers/         # start.py (кнопка Mini App, deep-links), payments.py (Stars)
server/               # FastAPI
├── auth.py           # валидация tg initData (HMAC), tg_user/tg_admin dependencies
├── game_config.py    # ВЕСЬ игровой баланс (крутить числа здесь)
├── game_logic.py     # энергия, XP, сезоны, дейлик, квесты, реф-майлстоуны
└── routers/
    ├── game.py       # /api/state, кликер, merge, уровни, ачивки, дейлик, квесты
    ├── meta.py       # /api/auth, рефералка+майлстоуны, промо, БП, магазин, канал
    └── admin.py      # /api/admin/* — промо, source, статистика, рассылка
db.py                 # SQLite + авто-миграция схемы
webapp/               # React + Vite + TS (Mini App), собирается в webapp/dist
main.py               # бот (polling) + API (uvicorn) + notifier одним процессом
```

## Запуск

1. Заполни `.env`:
   ```
   BOT_TOKEN = "токен от @BotFather"   # старый токен засветился — перевыпусти!
   ADMIN_ID = 1076078800
   BOT_USERNAME = "имя_бота_без_@"     # для реф- и source-ссылок
   WEBAPP_URL = "https://<твой-туннель-или-домен>"
   PORT = 8000
   CHANNEL_USERNAME = "имя_канала"     # опционально: награда за подписку в профиле
   ```
2. Собери фронт: `cd webapp && npm install && npm run build`
   (для реф-ссылок на фронте: `VITE_BOT_USERNAME=имя_бота npm run build`)
3. `python main.py` — поднимет бота и API на :8000, фронт раздаётся с корня.
4. Для теста в Telegram нужен HTTPS: `cloudflared tunnel --url http://localhost:8000`
   (или ngrok), полученный URL → в `WEBAPP_URL` **и** в настройки Mini App у @BotFather
   (/newapp или кнопка меню бота).

## Как что работает

- **Авторизация** — фронт шлёт `Authorization: tma <initData>`, сервер проверяет
  HMAC-подпись токеном бота. Сервер — источник правды: клики батчами с CPS-лимитом
  и энергией, все merge-ходы валидируются на сервере.
- **Рефералка** — ссылка `t.me/<bot>?startapp=ref_<user_id>`, взаимная награда
  начисляется обоим при первом входе приглашённого.
- **Source-ссылки** — создаются в админ-вкладке (`t.me/<bot>?startapp=src_<code>`),
  регистрации по ним видны в статистике.
- **Промокоды** — создаются в админ-вкладке, активируются в профиле (1 раз на юзера).
- **Stars** — `/api/shop/invoice` создаёт invoice-ссылку, фронт открывает её через
  `openInvoice`, начисление в `bot/handlers/payments.py` после `successful_payment`
  (идемпотентно по charge_id).
- **Админ-вкладка** появляется в Mini App автоматически, только если ты зашёл
  с аккаунта `ADMIN_ID`. Там же — рассылка всем игрокам (кнопка «Тест» шлёт превью себе).
- **Ежедневная награда** — попап при входе, стрик 7 дней (500→15 000 🍪),
  пропуск дня сбрасывает стрик. День считается по UTC.
- **Ежедневные задания** — 3 в день (выбор детерминирован датой, у всех одинаковые),
  награда: печеньки + XP батл-пасса. Прогресс копится в кликах/мерджах/покупках.
- **Сезоны** — 14 дней (`SEASON_EPOCH`/`SEASON_LENGTH_DAYS` в game_config).
  Лидерборд ранжируется по уровню, при равенстве — по `season_earned`;
  топ-10 в конце сезона получают призы
  (`SEASON_TOP_REWARDS`), батл-пасс и сезонный заработок сбрасываются. Ролловер
  ленивый — происходит при первом запросе после конца сезона, снапшот в `season_results`.
- **Реф-майлстоуны** — 3 друга → буст x2 на 24ч, 10 → эксклюзивный скин 👑,
  25 → Premium Пасс. Забираются в профиле.
- **Пуши** — `bot/notifier.py` раз в 15 мин проверяет всех: стрик сгорает / ферма
  упёрлась в кап / энергия полная. Максимум 1 пуш в ~20ч, активным (были онлайн
  < 3ч назад) не шлём, заблокировавших бота помечаем и пропускаем.
- **Золотая печенька** — сервер планирует появление (раз в 3-7 мин игры, живёт 12с),
  клиент рисует 🌟 и репортит тап. Эффекты: frenzy x7 на 25с или мгновенный бонус.
- **Комбо** — тапаешь без пауз (>=3 кликов/с) — множитель растёт до x2,
  пауза 4с сбрасывает. Считается на сервере по батчам.
- **Престиж** — после 10M заработанных: сброс прогресса (скины/друзья/ачивки/Stars
  сохраняются) за очки = sqrt(earned/1M), каждое очко даёт +2% ко всему доходу навсегда.
